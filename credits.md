# Credits

## Data Sources
This project uses publicly available datasets. Unless noted otherwise, datasets were downloaded from Our World in Data (OWID) Grapher with accompanying metadata.

- **AI training compute vs. parameters by researcher affiliation**  
  Source citation: Epoch AI (2025), with major processing by Our World in Data.  
  Local file: `ai-training-computation-vs-parameters-by-researcher-affiliation.csv`  
  Notes: This dataset is used directly in Chart 1 and has no local OWID metadata JSON file in this repo.

- **Child mortality rate**  
  Source citation: Gapminder (2015); UN Inter-agency Group for Child Mortality Estimation (2025) - processed by Our World in Data.  
  OWID chart URL: https://ourworldindata.org/grapher/child-mortality?v=1&csvType=full&useColumnShortNames=false  
  Local files: `child-mortality.csv`, `child-mortality.metadata.json`  
  Timespan: 1751-2023. Last updated: 2025-04-25.

- **Per capita primary energy consumption by source**

  Local files: `per-capita-energy-stacked.csv`, `per-capita-energy-stacked.metadata.json`
